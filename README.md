# TOEIC Part 5・6・7 トレーニング

TOEIC の Part 5（短文穴埋め）・Part 6（長文穴埋め）・Part 7（読解）と、単語・熟語の語彙問題をブラウザだけで学習できる単一HTMLアプリです。
ダウンロードして個人用にお使いください。

## 使い方

`index.html` をブラウザで開くだけ。インストール・サーバー・通信は不要で、スマホ表示（横幅720px基準）に最適化しています。

## 機能

- **Part 5｜短文穴埋め** — 4択の文法・語彙問題。
- **Part 6｜長文穴埋め** — 文脈に沿った空所補充。
- **Part 7｜読解** — 長文読解問題。
- **単語・熟語** — 英単語・熟語の意味を4択で答える1問1答（レベル別出題）。
- **学習の記録** — 日付・点数・カテゴリ別の弱点を記録。平均正答率やカテゴリ別正答率（弱点が上位）を表示し、記録をテキストとして書き出せます。

## データの保存

学習記録・進捗はブラウザの **localStorage** にこの端末のみ保存されます。サーバーには一切送信されません。別の端末・ブラウザとは共有されず、ブラウザのデータを消去すると記録も消えます。

## ファイル構成

| ファイル | 内容 |

## 技術

- 依存ライブラリなしの単一HTML（HTML/CSS/JavaScript）。
- ライト基調のレスポンシブUI。
